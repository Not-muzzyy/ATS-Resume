# ResumeAI — ATS Checker & Resume Builder

AI-powered resume analyzer and builder. Free to use, mobile-friendly, and built for everyone — technical and non-technical users.

## Features

- **ATS Score** (0–100) with keyword breakdown
- **Section feedback** — Summary, Skills, Experience, Education, Formatting
- **Missing keywords** detection from job description
- **Bullet point rewrites** with AI suggestions
- **Resume Builder** — fill a form, AI writes a professional resume
- **Role-specific tips** — get instant advice for any job
- **Download** report + generated resume

## Stack

- Python + Streamlit
- Google Gemini 2.0 Flash (free API)

## Local Setup

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy on Streamlit Cloud (Free)

1. Push this folder to a GitHub repo
1. Go to [share.streamlit.io](https://share.streamlit.io)
1. Connect your GitHub repo
1. Set main file: `app.py`
1. Deploy — done!

## Get Gemini API Key (Free)

1. Go to [aistudio.google.com](https://aistudio.google.com/app/apikey)
1. Sign in with Google
1. Click **Create API Key**
1. Paste it in the app when prompted
